Tool 002 · Site-hosted Video Automation

AI Edit Factory

Run AI edits from the web app: create a project, upload rights-approved songs and clips, click generate, then preview and download beat-synced vertical MP4s in the browser.

Launch the site app → Deployment guide →
01

Create online

Open the site app and make a project from the browser—no manual file placement required.

02

Upload media

Use the upload form for songs and source videos you own, licensed, or have permission to process.

03

Render on site

The API queues a backend worker job so ffmpeg renders MP4 edits on the site server, not on your laptop.

Runnable site command

cd ai-edit-factory
docker compose up --build

Open http://localhost:8000 for the full browser workflow. Generated files are still persisted under outputs/ for deployment storage.
