Tool 002 · Local Video Automation

AI Edit Factory

Drop in a song and source clips, then generate beat-synced 9:16 MP4 edits with a local ffmpeg-powered pipeline.

Read the full setup guide →
01

Add inputs

Place your track at inputs/song.mp3 and source videos in inputs/clips/.

02

Run locally

Activate your Python environment, install requirements, and run the app from this folder.

03

Export edits

Generated vertical videos are written to outputs/ as reusable MP4 files.

Runnable command

cd ai-edit-factory
python app.py --song inputs/song.mp3 --clips inputs/clips --num-edits 20

Expected output: beat detection, scene discovery, clip scoring, and rendered edit files in outputs/.
