pntn2 Complete Asset Bundle
===========================

This bundle compiles all currently available pntn2 mockups, extracted references, generated individual assets, batch 02 sprite assets, and the later crusader/instrument-separated asset set.

Folder guide
------------
00_mockups
  Original game mockups and scene concepts.

01_reference_extractions
  Rectangular crop extractions from the flattened mockup used as reference.

02_generated_assets_batch01
  First isolated PNG asset set generated from the mockup.

03_generated_assets_batch02
  Larger generated asset sheet plus individual extracted PNG assets.

04_crusader_instrument_separated
  Updated set where crusader bodies and instruments are separated.

Notes
-----
- Most files are PNGs intended as visual references for Codex.
- Some folders also contain contact sheets and manifests.
- The batch02 folder includes a more complete individual asset set extracted from a generated sprite sheet.
- The crusader/instrument-separated folder is the latest pass for modular character + instrument assembly.
